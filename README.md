# AI 小说创作助手

一个基于纯前端技术的AI辅助小说创作工具，可以托管在GitHub Pages或其他静态网站平台。

## ✨ 特性

- 🎨 **精美UI设计** - 采用现代化渐变背景、流畅动画和沉浸式交互体验
- 🤖 **AI智能辅助** - 支持OpenAI及兼容API，智能生成大纲、细纲和章节内容
- 📝 **完整创作流程** - 从构思到成稿的全流程支持
- 💾 **灵活存储** - 支持File System API本地保存，自动浏览器缓存备份
- 📤 **导出功能** - 支持导出为JSON项目文件或TXT文本文件
- 🎯 **元数据管理** - 自动追踪角色发展、情节线索和伏笔设置
- 📱 **响应式设计** - 完美支持桌面和移动设备

## 📖 使用指南

### 1. 配置AI服务

首次使用需要配置AI服务：
- **Base URL**: OpenAI API地址（如 `https://api.openai.com/v1`）
- **Model Name**: 模型名称（如 `gpt-4o`, `gpt-3.5-turbo`）
- **API Key**: 您的API密钥

### 2. 创作流程

#### 第一步：输入小说信息
- **小说描述**（必填）：描述故事背景、主要情节、角色设定
- **参考内容**（选填）：提供参考的小说片段或设定资料
- **文风要求**（选填）：如"古风"、"现代都市"、"科幻硬核"等

#### 第二步：生成并确认大纲
- AI根据您的输入生成整体大纲
- 您可以手动编辑和调整
- 确认后进入下一步

#### 第三步：生成并确认细纲
- AI将大纲扩展为详细的细纲
- 包含章节划分和详细情节规划
- 可以手动修改后确认

#### 第四步：章节创作（循环进行）
对每个章节依次进行：

1. **确认章节摘要** - 查看和编辑章节标题与摘要
2. **生成章节内容** - AI根据摘要生成完整章节
3. **编辑章节内容** - 审阅和修改生成的内容
4. **更新元数据** - AI自动提取：
   - 角色信息和变化
   - 情节发展
   - 伏笔与线索
5. **确认元数据** - 审核和补充元数据
6. **完成章节** - 进入下一章节

#### 完成创作
所有章节完成后，可以导出完整小说为TXT文件。

## 💾 数据存储

### 自动保存
- 每30秒自动保存到浏览器本地存储
- 数据变更时实时缓存
- 关闭页面后数据保留

### 手动操作
- **导出项目**：导出为JSON格式，包含所有数据
- **导入项目**：从JSON文件恢复项目
- **导出小说**：导出为纯文本TXT文件

### File System API
在支持的浏览器中（Chrome、Edge等），可以直接保存到本地文件系统。

## 🎨 UI特色

- **渐变背景**：柔和的紫色到蓝色到粉色渐变
- **流畅动画**：淡入淡出、滑动、缩放等过渡效果
